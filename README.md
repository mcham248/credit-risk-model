# credit-risk-model
Credit risk default prediction model using Logistic Regression, Random Forest and XGBoost (AUC = 0.85).

# Credit Risk Default Prediction

Machine learning project predicting borrower default using the "Give Me Some Credit" dataset.

## Models
- Logistic Regression
- Random Forest
- XGBoost

Best model: **XGBoost (AUC = 0.85)**

## Features
- Revolving credit utilisation
- Debt ratio
- Late payment history
- Income
- Number of loans

## Results
The model effectively separates high-risk and low-risk borrowers and can be used to simulate lending approval policies.

## Tools
Python, Pandas, Scikit-learn, XGBoost, Matplotlib
